# Installation

## Development branch

1) download & install [bazaar](http://bazaar-vcs.org/) - instructions at [bazaar download page](http://bazaar-vcs.org/Download)

2) run command:

    bzr branch http://bazaar.launchpad.net/~karol.tarcak/webpy/webpy.sessions.branch webpy

3) enjoy
